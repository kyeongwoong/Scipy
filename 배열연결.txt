# 배열 연결(concatenate)  
# 행의 수나 열의 수가 같은 두개 이상의 배열을 연결하여
# 더 큰 배열을 만들 사용하는 방법을 말한다.

# hstack : 행의 수가 같은 두 개 이상의 배열을 옆으로 연결하여 열의 수가 더 많은
#	배열을 만든다. 연결할 배열은 하나의 리스트에 담아야 한다.
# vstack : 열의 수가 같은 두 개 이상의 배열을 위아래로 연결하여 행의 수가 더 많은
	배열을 만든다. 연결할 배열은 마찬가지로 하나의 리스트에 담아야 한다.
# dstack : d는 깊이(depth)를 의미한다. 즉, 행이나 열이 아닌 깊이 방향으로 
#	배열을 합친다.
# stack : dstack 명령을 확장한 것, 배열 두개가 겹치므로 연결하고자 하는 배열들의 
#	크기가 모두 같아야 한다. 인수로 axis를 사용할 수 있다. 디폴트값은 0이다.
# r_ 메소드: hstack 과 비슷한 명령, 배열을 좌우로 연결한다. ( )를 사용하지 않는다.
#	인덱싱과 같이 대괄호[ ]를 사용한다. 이러한 특수 메소드를 
#	인덱서(indexer)라고 한다.
# c_ 메소드: 배열의 차원을 증가시킨 후 좌우로 연결한다. 
#	따라서, 1차원 배열을 연결하면 2차원 배열이 된다.
# tile : 동일한 배열을 반복해서 연결할 때 사용한다.
