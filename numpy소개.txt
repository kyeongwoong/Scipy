numpy 설치:
pip3 install numpy

numpy 가져오기:
import numpy as np

Numpy : 고성능 과학계산을 위한 데이터분석(수치분석) 패키지
	- 벡터 산술연산
	- 다차원 배열 (ndarray)
	- 표준 수학 함수
	- 선형대수, 난수

numpy배열 : 같은 종류의 데이터를 담는 자료형이다.(동일한 타입의 값들을 갖는다.)
	  
rank : 배열의 차원을 의미한다. rank를 알아보기 위해서는 ndim속성을 사용한다.
shape : 각차원의 크기를 튜플로 표시한 것 (행, 렬) 
dtype : 배열에 저장된 자료를 알려주는 객체
	사용예> ndarray.dtype.name 또는 ndarray.dtype

         * numpy의 자료형 : float, complex(복소수), int, bool, string, object(객체)	

# ndarray : 다차원 배열 객체
   다차원 배열 생성 함수 
	- array()	
	- arange().reshape() 
	- zeros() : 0이 들어 있는 배열을 생성
	- ones() : 1이 들어 있는 배열을 생성
	- full() : 모든 값이 특정 상수인 배열을 생성
	- eye() : 단위 행렬 생성

# 배열 인덱싱
  - 슬라이싱 : 파이썬 리스트와 유사하게, numpy에서도 슬라이싱이 가능	
  
      * 슬라이싱된 배열은 원본 배열과 같은 데이터를 참조하기 때문에, 
        슬라이싱된 배열을 수정하면 원본 배열도 역시 수정된다.	 	

      * numpy배열을 슬라이싱하면, 그 결과는 언제나 원본 배열의 부분배열이다.
        그러나, 정수 배열 인덱싱을 하는 경우에는 원본과 다른 배열을 만들 수 있다.
        
      * 배열 인덱싱(array indexing)은 팬시 인덱싱(fancy indexing)이라고도 한다.
        - 배열 인덱싱의 유용한 기능 중의 하나는 행렬의 각행에서 하나의 요소를
          선택하거나 바꾸는 기능이다. 

# 배열 연산
  - 기본적인 수학함수는 배열의 각 요소별로 동작한다. 그리고 연산자를 통해 동작
    하기도 하고 numpy함수 모듈을 통해서 동작하기도 한다.
